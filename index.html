What is blazug.js ?


Blazug is a lightweight and simple-to-use vanilla js library that displays the browser console inside an HTML page. It is helpful to debug mobile website. Blazug also provides some controls to quickly display some text or trigger action for testing purposes.

Blazug also integrates seamlessly inside Blazor application.

Install



Two lines

  1. Insert blazug stylesheet anywhere inside your HTML <head>
    <link rel='stylesheet' href="https://cdn.jsdelivr.net/gh/QwotApp/TestBedBlazug@2.2.4/blazug.css"/>

  2. Insert blazug script at the beginning of the HTML <body>
    <script src="https://cdn.jsdelivr.net/gh/QwotApp/TestBedBlazug@2.2.4/blazug.js"></script>

Then add some console.log() and click on the bug to watch the console.

Display mode


To change the display mode at startup, assign one of the following values to the startup attribute of the blazug's script tag.

  1. normal: Show only the bug button. (default behavior)

  2. hidden: Hidden at startup, nothing displayed.

  3. console: The console window is visible at startup.

  4. controls: The controls window is visible at startup.


The display mode can also be changed dynamically by calling the function blazug.display(mode)

             
      blazug.display("controls");

Copy Log to clipboard


Click on the log inside the console window to copy it to clipboard.

Download Logs


To download the logs call the function blazug.downloadLogs()

             
              blazug.downloadLogs();

Controls



Display Text


blazug.displayText("controlId","text","minsize") show a text inside the controls view. It can be called multiple time with the same controlId to update the text

  1. controlId - string,required : unique controlId, will be displayed as a tip over the text control.

  2. text - string,required : the text string to be displayed/updated.

  3. minsize - string,optional : minimum width of the control, can be content (default), small, medium, large, max.

Button


blazug.createButton("controlId","text","minsize",callback) display a button that triggers a callback when pressed.

  1. controlId - string,required : unique controlId, will be displayed as a tip over the text control.

  2. text - string,required : the text string to be displayed/updated.

  3. minsize - string,optional : minimum width of the control, can be content (default), small, medium, large, max.

  4. callback - function,required : function called when the button is clicked.

             
      blazug.createButton("logs", "Download", "content", (controlId, element) => {
        blazug.downloadLogs();
      });

Switch


blazug.createSwitch("controlId","textOn","textOff",initialState,callback) display a switch that triggers a callback when state has changed.

  1. controlId - string,required : unique controlId, will be displayed as a tip over the text control.

  2. textOn - string,optional : text label when true. default is On.

  3. textOff - string,optional : text label when false. default is Off.

  4. initial - boolean,optional : initial statedefault is false.

  5. callback - function,required : function called when the state has changed.

             
      blazug.createSwitch("toggle", "True", "False", true, "content", (controlId, state, text, element) => {
        alert("switch '" + controlId + "' state is '" + text + "' (" + state + ").");
      });

Radio Buttons


blazug.createRadio("controlId","textOn","textOff",initialState,callback) display a switch that triggers a callback when state has changed.

  1. controlId - string,required : unique controlId, will be displayed as a tip over the text control.

  2. buttonsText - array,required : array of buttons text.

  3. initial - boolean,optional : initial index of active button.default is 0.

  4. callback - function,required : function called when the state has changed.

             
        blazug.createRadio("theme", ["Light", "Dark", "Black"], 1, (controlId, index, text, element) => {
          alert("radio '" + controlId + "' state is '" + text + "' (index: " + index + ").");
        });
