What is blazug.js ?

Blazug is lightweight and simple to use vanilla js library that displays the browser console inside an html page. It is especially helpfull for debugging on mobile website. Blazug also provides some controls to quickly display some text or trigger an action for testing purpose.

Blazug integrates seamlessly inside Blazor webassembly application. For more information about the nuget package, go to BlazugLib on gihub.

Install

Two lines

  1. Insert blazug stylesheet <link rel='stylesheet' href='./blazug.css'/> any where inside your html header

  2. Insert blazug script <script src="./blazug.js"></script>at the beginning of the body.

Then add some console.log() and click on the bug button to watch the console.

HTML5 doctype

Enforce standards mode and more consistent rendering in every browser possible with this simple doctype at the beginning of every HTML page. In keeping with the suggested syntax, keep it lowercase.
